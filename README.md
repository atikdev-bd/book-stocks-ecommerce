## Key specification:

- express js.
- using nodeJS.
- use react-Query.
- use json web token.
- set a page for 404 not found.
- use stripe payment getway.
- Use react hot toast.
- Using useForm with react-router-dom.
- Using react spinner.
- Using mongoDB .
- Using firebase.
- Using Environment variable server side and client side.
- react-router-dom, tailwind, daisyUI etc used for this site.
